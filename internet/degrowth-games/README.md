# degrowth-games

## Quickstart

Run directly with nodejs (nodejs knows [how to strip types](https://nodejs.org/api/typescript.html#type-stripping)).
There is no need to run `npm install` as there is no third party dependency to run the code:

```
node index.ts
```

Without any argument, an help message will be displayed

## Optional typechecking

If you want to check that types are correct, you can run typescript as the typechecker:

```
npm install
npx tsc
```
